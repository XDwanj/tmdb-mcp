# 测试 MCP 服务器

来源：https://github.com/modelcontextprotocol/go-sdk/blob/main/examples/server/auth-middleware/README.md

用于运行单元测试、基准测试和生成 MCP 服务器代码覆盖率报告的 Go 命令。

```bash
# Run all tests
go test -v

# Run benchmark tests
go test -bench=.

# Generate coverage report
go test -cover
```
