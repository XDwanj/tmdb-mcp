# Go Resty TraceInfo 方法

Source: https://github.com/go-resty/docs/blob/main/content/docs/new-features-and-enhancements.md

本节记录了 Go Resty 库中用于格式化和访问跟踪信息的 GetTraceInfo 方法。它包括将跟踪信息作为字符串或 JSON 获取的方法。

```APIDOC
TraceInfo Methods:

String() string
  - 将跟踪信息作为格式化字符串返回。

JSON() string
  - 将跟踪信息作为 JSON 字符串返回。
```
